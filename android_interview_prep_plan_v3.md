# Android Developer Interview Prep — 12-Week Detailed Plan (v3: Bangladesh Week + Class Schedule Edition)

A day-by-day execution plan built on top of the 3-month blueprint, so you never have to ask "what do I actually do today?"

**What's new in v3:**
- The week now runs **Saturday → Friday**, matching the Bangladesh week. Friday (national holiday) is the rest day — which lines up naturally, since the old rest day was already the last day of the cycle.
- Your **Summer '26 class routine** is built into the week. Each weekday row now shows when you're in class and when you're not.
- A **🎓 Course Study** block is reserved on every class day and on the free weekend days, so coursework doesn't get squeezed out by Android prep — or vice versa.
- On heavy class days, the DSA/Deep Dive/Capstone/Reverse-Engineering blocks are **compressed**, not skipped. Full-length blocks live on your free days (Sat, Sun), where you have the most room.

## 🛡️ Anti-Avoidance Protocol (unchanged — still applies every day)

1. **The 2-minute Start Trigger, no decision required.** When the scheduled block arrives, don't decide whether to start — just do that day's tiny trigger.
2. **The Browsing Box.** Urge to research around a problem instead of attacking it? 10-minute timer, then close the tabs and go straight back to the problem.
3. **Weeks 1–2: track "did I start," not "did I finish."** Showing up and doing the trigger counts as a win.
